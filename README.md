Um Chat pessoal que usa o poder do GPT e pode receber seus dados/arquivos como parâmetro e ser um assistente completo de programação.

Atenção: é necessário criar a sua própria key para usar a API "openai.api_key", feito a partir do sistema da OpenAi, no site deles (https://platform.openai.com/api-keys).
Há uma cobrança por parte da OpenAI pelo uso dessa API, que escala com a quantidade de tokens enviados para o GPT.
